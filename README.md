Contains the sources for csl.name
=================================

You can either build on your local machine from source/ (read the instructions
there), or you can do use Docker.

To build the image `csl.name`:

    $ make docker-build

To build the files:

    $ make docker-build

To serve them:

    $ make docker-serve

To open a shell:

    $ make docker-bash
